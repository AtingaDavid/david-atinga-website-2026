## 🔄 How to Update the Site (from your phone)

### Option A — GitHub Web Editor (easiest)
1. Open this repo on github.com on your phone
2. Tap `index.html` → tap the ✏️ pencil icon (Edit)
3. Make your changes
4. Scroll down → tap **"Commit changes"**
5. Netlify auto-deploys in ~30 seconds ✅

### Option B — Upload new files (for photo/CV/PDF)
1. Open this repo on github.com
2. Tap **"Add file" → "Upload files"**
3. Choose your file (photo.jpg, resume.pdf, etc.)
4. Tap **"Commit changes"**
5. Netlify auto-deploys in ~30 seconds ✅

### Option C — GitHub Mobile App
Install the **GitHub Mobile** app (iOS/Android) for a smoother editing experience on your phone.
